# Review Docs

В этой папке лежат review-related документы: handoff, detailed review, targeted review reports и связанные review specs.

Правило:

- review-документы полезны для текущей проверки и follow-up work;
- они не считаются каноническим source of truth на уровне репозитория;
- канонические runtime и architecture contracts остаются в корневых canonical docs и тематических canonical folders.
